# TimeLeap Memo

「昨日の自分」が今に割り込む、時間軸メモの試作アプリ。

## Download
exeはこちら：
- Releases ページからダウンロード(win10/win11)

※手書きなのでペンタブ推奨

## What is this?
- メモが時間順に再構成される
- 過去の記録が“再生”される感覚
- 思考を「線」ではなく「層」として扱うUI
- ページ移動の概念がなく、連続的に移り変わっていくため、無限に書くことが出来ます。

## なぜタイムリープ？
- 時間軸を戻してその時の自分の思考プロセスを可視化し、それを客観視する体験を、過去に戻る“タイムリープ”と解釈している。
- そこから書き足して分岐した新たな思考の枝を多世界解釈の平行世界であると考えている。

## How to use
1. exeを起動
2. メモを取る
3. タイムラインを操作

ショートカットキー
- C: クリア、すべて消えます。
- B: マルチバースパッド、X軸でタイムライン操作、Y軸で並行世界の選択ができます。
- N: ナウ、現在時刻、つまり今書き進めている最後の時間まで戻れます。
