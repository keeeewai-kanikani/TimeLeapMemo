TimeLeapMemoとは、タイムリープ、多世界解釈の概念を取り入れた3次元メモ帳アプリである


2023夏 東京でメモをとりまくる。この時メモ帳とニ〇ニコ動画時間操作の融合を模索し始める。

2024夏 日記を書き起こすことで頭がすっきりし、生産性が上がることを実感

2024秋 製図の教科書の前書きに感銘を受け、思考の製図規格化という概念を思いつく。

2025/11/5 考えるためのメモについての研究の為このアプリの作成を開始。

2025/11/8 v1.0.0完成


ページ移動の概念がなく、連続的なタイムラインで文字が薄れ、移り変わっていくため、無限に書くことが出来ます。


ショートカットキー
Cでクリア、すべて消えます。
Bでマルチバースパッド、X軸でタイムライン操作、Y軸で並行世界の選択ができます。
Nでナウ、現在時刻、つまり今書き進めている最後の時間まで戻れます。
